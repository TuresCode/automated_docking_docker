Automated Docking Pipeline

Please upload sdf and pdb files

Instructions: You can upload multiple pdb files and multiple sdf files containing either one or mutliple ligands in 3D to perform docking with it. You can get 3D sdf files for e.g. from Pubchem. By selecting GPU Vina you can boost the docking process (currently not available). Selecting either metal containing or NADP cofactor is trying to identify the NADP or Metal and generating a docking Box around it in all pdb's, if you select one of these you don't need to provide center coordinates. Usign align, all your pdbs are going to be aligned before the docking. Selecting fixed torsion sets the torsion angle of your susbtrate to fixed which comes in handy when having aromatic planar ligands. Please, be aware that when docking multiple pdbs and sdf it might take a while for the uploade. In this case just wait.

  • ■ Where can I find my center coordinates? Open PyMOL and select an atom in the active site: enter print(cmd.get_coords('sele')) into the command line.
  • ■ How to get a sdf file? Pubchem by downloading the 3D sdf file or e.g. ChemDraw, draw molecule, select and Get 3D Model under Edit or by copying to Chem3D. Then save as .sdf file (V3000)
  • ■ How does the align work? Alignment is done on the first pdb structure (alphabetically). Add a 1_ as prefix to the file name to use it as reference.
